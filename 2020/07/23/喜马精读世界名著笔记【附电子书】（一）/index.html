《变形记》

推广宣传语：一觉醒来变成了虫，生活怎么继续过下去？
分类：外国文学
国家：捷克
作者：卡夫卡

“今天早晨一睁眼，视野竟完全不如以往，我居然变成一只大甲虫？”

格里高尔一觉醒来后，发现自己不再是人，成了一只又大又恶的害虫。他不仅丢了工作，原本家人平静的生活因此波折不断，格里高尔成了一家的负担。最后，为了让家人不再受苦，格里高尔选择死亡。

《变形记》作为20世纪最伟大的小说之一，对世界文学和中国当代文学都产生了深远影响，这部小说被多次翻拍成电影及戏剧，也是众多作家创作的泉源。德国著名作家托马斯·曼将此列为“最值得一读的世界文学名作”。

关于作者

弗兰兹·卡夫卡（Franz Kafka，1883-1924），出生在奥匈帝国的德语小说家，20世纪最具影响力的作家之一，被誉为西方现代主义文学及表现主义文学的先驱。生前代表作品有《变形记》、《审判》和《城堡》。

本书金句

如果现在一切宁静、一切舒适、一切满足都要恐怖地宣告结束的话，情况又会怎么样呢？

清晨那会儿，所有的门全锁着，大家都想进来见他，现在他开开了一扇门，却又谁也不来了。

除了等待，他现在没有别的事可做；最后在绝望中，他觉得整个房间已经开始绕着他旋转起来。

每逢格里高尔又身处黑暗之中而隔壁妇人们涕泣交流的时候，他便觉得背上的伤口好似重新疼痛了起来。

他怀着深情和爱意回忆他的一家人，他的鼻孔呼出了最后一丝微弱的气息。

小节标题

1.活在资本社会中的社畜幻想故事
2.被朝九晚五的工作折磨的现代人：一觉醒来发现自己变成了虫
3.朝不保夕的饭碗：为什么现代人活得越来越怂？
4.当你失去了经济生产的能力，你还能活得人模人样吗？
5.家庭关系的异化：阴森恐怖的结局
6.丧文化中的正能量

《罗杰疑案》

推广宣传语：侦探小说女王的成名代表作，你绝对猜不到的杀人凶手
分类：文学
国家：英国
作者：阿加莎·克里斯蒂

关于本书：

“阿加莎始终认为人性本恶，在她的作品中，源于人性的犯罪动机普遍存在，于是凶手可以在拥有罪犯身份的同时伪装成任何人。”

自从弗拉尔斯太太死后，金丝艾伯特最有钱的大财主就只剩下罗杰了。然而在一个寒冷刺骨的夜晚，罗杰在自己的庄园中被人一刀毙命。客人、家人、佣人，当晚庄园里人数着实不少，而他们每一个人都欲盖弥彰的隐藏着自己的秘密。可是这一切，早就被充满智慧的大侦探波洛洞悉了真相……

这部小说，阿加莎用极度冷漠的语言和描写解剖了人性，她在写作的行云流水中寻找人生哲学。《罗杰疑案》在美国推理作家协会(MWA) 票选出来的“史上最经典的百部推理小说排行榜”中，高居第12名。这部作品是阿加莎的成名代表作，从此她正式一步步踏上“侦探女王”的宝座。

关于作者：

阿加莎·克里斯蒂（Agatha Christie，1890-1976），英国侦探小说家、剧作家。阿加莎是世界三大推理文学宗师之一，毋庸置疑的侦探女王，侦探界的“流量琼瑶”。据统计，她的一生共撰写了80部侦探小说，被翻译成100多种文字、畅销150多个国家，她的作品销量仅次于《圣经》和《莎士比亚戏剧集》。

本书金句：

动用一下你的小小灰色细胞。

女人能够下意识地观察到许多细节的问题，她们并不知道其中的原因，但她们的下意识会把这些细节组合在一起——人们把这种现象称之为结果自觉。

一旦你心底暗暗坚信的事情被别人戳穿，就难免恼羞成怒、矢口否认。

我们生活中的大多数事情不是愚蠢的就是为了虚荣。

世上什么怪事都有。我们整天忙忙碌碌地工作就是为了达到某一个目标，一旦目标达到了，我们就会发现我们失去的正是每天的工作。

小节标题

1.暗潮涌动的谋杀之夜
2.凶手就在他们之中
3.是谁撒了谎？又是谁选择了沉默？
4.揭开被隐藏的秘密
5.无法解释的死亡电话
6.被忽略的关键线索
7.真正的凶手竟然是“自己”

《城堡》

推广宣传语：真理近在咫尺，但我们永远触摸不到
分类：文学
国家：奥地利
作者：弗兰兹·卡夫卡

关于本书

“这个人为了他的信仰而从他的矛盾中搜寻理由，以便在他的有效益的绝望中能够有所希望，这个人把生存称之为一种对于死亡的可怕的准备。”

《城堡》是一个简单又深奥的寓言：土地测量员K受命赴某城上任，不料却受阻于城堡大门外。主人公K同城堡当局围绕能否进入城堡之事展开了持久烦琐的拉锯战。城堡就位于眼前一座小山上，可它可望而不可即。K直到最后也没有进入城堡，也没能见到城堡的主人。

关于作者

弗兰兹·卡夫卡（Franz Kafka 1883-1924），犹太人，奥匈帝国统治时期的捷克德语小说家，生前鲜为人知，死后却蜚声世界文坛，被称为“作家中的作家”，著有《变形记》《审判》《城堡》等作品，作品以荒诞的情节和深刻的寓意著称。

本书金句

  1. K抵达的时候，夜已深了。村庄被厚厚的积雪覆盖着。城堡山笼罩在正在雾霭和夜色中毫无踪影，也没有一丝灯光显示出巨大城堡的存在。K久久站立在有由大陆通向存在的木桥上，仰视这似乎虚无飘渺的空间。

  2. 他们宠着他，以此削弱他的力量，排除他进行各种斗争的可能性，让他去过一种非官方的、不明不白的、忧郁的异乡人的生活。

  3. 努力想得到什么东西，其实只要沉着镇静、实事求是，就可以轻易地、神不知鬼不觉地达到目的。而如果过于使劲，闹得太凶，太幼稚，太没有经验，就哭啊，抓啊，拉啊，像一个小孩扯桌布，结果却是一无所获，只不过把桌上的好东西都扯到地上，永远也得不到了。

  4. 不要失望，甚至对你并不感到失望这一点也不要失望，恰恰在似乎一切都完了的时候，新的力量来临，给你以支柱，而这正表明你是活着的。

  5. 我是自由的，那就是我迷失的原因。

小节标题

  1. 一部无所不包的神秘作品

  2. 荒诞的世界，符号化的人

  3. 城堡与真理，伯爵与上帝

  4. 艺术家的自我寓言

《暗店街》

推广宣传语：对记忆的书写，是为了抵抗遗忘；用记忆的艺术，书写黑暗的历史时代；用记忆的艺术，捕捉到二战法国被占领期间普通人的生活
分类：文学
国家：法国
作者：帕特里克•莫迪亚诺

关于本书

这确实就是我自己的生活呢？还是我潜入了另一个人的生活？

二战后的巴黎，失去记忆的私家侦探居伊•罗朗为了寻找自己失去的记忆，对不同身份的人物进行调查。而那些不断被牵扯出来的记忆碎片，可能是居伊自己的人生断片，也可能是其他人的。居伊孜孜不倦地寻访着那些记忆的证人。不约而同的，每个人都会把记忆引向那个遥远而黑暗的历史时代……

关于作者

帕特里克•莫迪亚诺（Patrick Modiano 1945— ），法国著名小说家，是法国评论界一致公认的当今法国最有才华的作家之一。代表作有《星形广场》、《夜巡》、《环城大道》、《暗店街》等。莫迪亚诺曾获得法国龚古尔文学奖、奥地利国家欧洲文学奖，并于2014年斩获诺贝尔文学奖。

本书金句

1.我已经度过了自己的一生，如今我只是一个在周末夜晚的和暖空气里游荡着的鬼魂。为何我还要重新连接起已经断裂的纽带，寻觅已经被砌死的道路？

2.其实我们大家都是沙滩人。沙子只把我们的脚印保留几秒钟。

3.我在心里一遍遍重复着出生时别人给我起的名字，在我的一生中，有很长一段时间人家都用这个名字来称呼我。一提起这个名字，大家就会想起我的面孔。佩德罗。

4.生活中重要的不是未来，而是过去。

5.直到现在，我还是觉得一切都是那么混乱无序，那样破碎不全……生活大概就是这样的吧。这确实就是我自己的生活呢？还是我潜入了别人的生活？

小节标题

1.他是又一个普鲁斯特吗？

2.这是一部非典型侦探小说，谜底是记忆

3.用记忆的艺术，书写黑暗的历史时代

4.我们都是“海滩人”

5.书写记忆，是为了抵抗遗忘

《罗生门 》

推广宣传语：每个人都会为了私欲撒谎，会在自己的谎言下，心安理得地作恶
分类：文学
国家：日本
作者：芥川龙之介

关于本书

一个雨夜，有一个被主家解雇的仆人，正巧走到一个到处都堆满死尸的地方，也就是叫做罗生门的地方避雨。此时的仆人，正在经历着，到底是饿死还是去做强盗的心理挣扎。为了能找个睡安稳觉的地方，仆人壮着胆子，走进了罗生门……

一场竹林中的凶杀案，一个过路的强盗杀死了一个武士，还强暴了武士的妻子。一个过路的樵夫发现了武士的尸体，慌慌张张地报了案。吊诡的是，对于事情的经过，每个当事人各有说辞，并且每个人的说法竟然都没有破绽……

由黑泽明执导的《罗生门》，现已成日本经典电影之一，其原著正是芥川龙之介的《竹林中》和《罗生门》。

关于作者

芥川龙之介（1892-1927），日本新思潮派代表作家，与森鸥外、夏目漱石被称为20世纪前半叶日本文坛上的三巨匠。其作品以短篇小说为主，多为历史题材，情节新奇甚至诡异，以冷峻的文笔和简洁有力的语言让读者关注到社会丑恶现象。代表作《罗生门》《地狱变》等。

本书金句

  1. 为了摆脱毫无指望的困境，已经没时间去考虑选择什么手段了。

  2. 撒谎是人之本性，在大多数时间里我们甚至都不能对自己诚实。

  3. 我们必须在同人生的抗争中学习对付人生。如果有人对这种荒诞的比赛愤愤不平，最好儘快退出场去。自杀也确乎不失为一条捷径。但决心留在场内的，便只有奋力拼搏。

  4. 武器本身不足为惧，恐惧的是武将的武艺；正义本身不足为惧，恐惧的是煽动家的雄辩。

  5. 人们太脆弱了才撒谎，甚至是对自己撒谎。

小节标题

1.黑泽明的罗生门和芥川龙之介的罗生门

2.罗生门里，一个强盗的心路历程

3.借古人的故事，展现今人的心灵

4.时代的不安和人性的深渊

《远山淡影》

推广宣传语：新科诺奖得主一鸣惊人的处女作，回忆是不可靠的
分类：文学
国别：英国
作者：石黑一雄

关于本书

石黑一雄的小说，以其巨大的情感力量，发掘了隐藏在我们与世界联系的幻觉之下的深渊。

《远山淡影》是石黑一雄的处女作，出版于1983年。小说讲述了移民英国的日本女人悦子，在丧女之后陷入迷雾重重的回忆中。她想起了在战后的长崎与一对母女的短暂交往，但那段回忆亦真亦幻，漏洞百出。剧终，我们会触及到无法测量的情感深渊。

关于作者

石黑一雄，日裔英国小说家，1954年出生于日本长崎。1982年开始发表小说，代表作有《远山淡影》、《长日留痕》、《小夜曲》等。石黑一雄作品数量不多，但在国际上享有盛誉，他和鲁西迪、奈保尔一起被称为英国文坛的“移民三杰”。2017年10月，石黑一雄获得诺贝尔文学奖。

本书金句

  1. 英国人有一个奇特的想法，觉得我们这个民族天生爱自杀，好像无需多解释。

  2. 很多女人被孩子和讨厌的丈夫捆住手脚，过得很不开心。可是她们没有勇气改变一切。就这么过完一生。

  3. 回忆，我发现，可能是不可靠的东西；常常被你回忆时的环境所大大地扭曲……

  4. 我们也许在战争中失去了很多，但是还有那么多盼头。

  5. 只有一些空房间，没别的了。我可以找一间坐着，然后慢慢变老。除此之外什么都没有。

小节标题

1.2017年诺贝尔文学奖再爆冷门，石黑一雄到底是谁？

2.现实：英国南部，日本女人和她上吊自杀的女儿

3.回忆：日本长崎，一个单亲母亲溺死了女儿的猫

4.谜底：无法承受的往事，让一切回忆都不可靠

5.深渊：灾难改变了一切，和她们一样痛苦的女人，遍地都是

《瓦尔登湖》

推广宣传语：感动全世界亿万读者，一本让人心归宁静的经典之作
分类：文学
国家：美国
作者：亨利·大卫·梭罗

关于本书

“不必给我爱，不必给我钱，不必给我名誉，给我真理吧。”

1845年7月4日，梭罗开始了一项为期两年的试验，移居到离家乡康科德城不远的瓦尔登湖畔次生林，开始过一种简单的隐居生活，并在湖畔写下了这本《瓦尔登湖》。

它是美国历史上影响最广、受众最多的散文作品，至今仍是美国最受欢迎的非虚构类图书。著名评论家伊拉·布鲁克评价说，“在过去的一百年里，瓦尔登湖已经成为美国文化中纯洁天堂的同义词。”印度的“圣雄”甘地也是梭罗的忠实读者，甚至宣称梭罗是“美国有史以来最伟大的贤人”。

关于作者

亨利·大卫·梭罗（Henry David Thoreau，1817-1862），美国作家、诗人、哲学家、废奴主义者、超验主义者，也曾任职土地勘测员，最知名的作品包括《瓦尔登湖》和《公民不服从》。

本书金句

  1. 我愿意深深地扎入生活，吮尽生活的骨髓，过得扎实，简单，把一切不属于生活的内容剔除得干净利落，把生活逼到绝处，用最基本的形式，简单，简单，再简单。

  2. 我们天性中最优美的品格，好比果实上的粉霜一样，是只能轻手轻脚，才得保全的。

  3. 我步入丛林，因为我希望生活得有意义，我希望活得深刻，只去面对生命最本质的那些事实，然后从中学习，以免让我在生命终结时，发现自己从来没有活过。

  4. 一个人若能自信地向他梦想的方向行进，努力经营他所想望的生活，他是可以获得通常还意想不到的成功的。他将要越过一条看不见的界线，他将要把一些事物抛在后面；新的、更广大的、更自由的规律将要开始围绕着他，并且在他的内心里建立起来；或者旧有的规律将要扩大，并在更自由的意义里得到有利于他的新解释，他将要拿到许可证，生活在事物的更高级的秩序中。他自己的生活越简单，宇宙的规律也就越显得简单，寂寞将不成其为寂寞，贫困将不成其为贫困，软弱将不成其为软弱。

  5. 多余的财富只能够买多余的东西，人的灵魂必需的东西，是不需要花钱买的。

小节标题

1.瓦尔登湖已经成为美国文化中纯洁天堂的同义词

2.孤独是最好的陪伴

3.极简生活只需要这两样东西

4.瓦尔登湖不是隐匿，而是冒险

5.像走进瓦尔登湖那样离开它

《莎士比亚十四行诗》

推广宣传语：相信爱和诗具有永恒的生命和价值
分类：诗与散文
国家：英国
作者：威廉·莎士比亚

关于本书：

莎士比亚被誉为“人类最伟大的戏剧天才”，他为世人留下了37部伟大的戏剧作品。除了这些戏剧作品之外，诗歌在莎士比亚的全部创作中也占有十分重要的位置，尤其是他的十四行诗，达到了同类诗歌创作的顶峰，而且是莎士比亚生前出版的最后一部非戏剧类作品。

由著名翻译家梁宗岱先生翻译的这本《莎士比亚十四行诗》，收录了莎士比亚创作的全部154首十四行诗。莎士比亚在这些结构严谨的抒情诗里延续了他一贯的天马行空般的想象力，灌之以丰沛的情感和深沉的思考，对包括爱情、时间、生育、衰老、死亡等等在内的文学母题进行了深入的探讨，是打开和进入莎士比亚内心世界的一把钥匙。

关于作者：

威廉·莎士比亚（William Shakespeare，1564-1616），欧洲文艺复兴时期英国著名的戏剧家、人文主义思想家和诗人，其主要作品包括《哈姆雷特》、《李尔王》、《奥赛罗》、《麦克白》、《罗密欧与朱丽叶》等30余部戏剧、154首十四行诗和两首长诗。莎士比亚的作品对后来的世界文学产生了极为广泛和深远的影响，这些作品属于人类历史上最为宝贵的文学财富之一。

本书金句：

我的爱在我诗里将万古长青。

你的仪表属于我的眼睛，而我的心占有你心里的爱情。

每人都只有一个，每人，一个影；你一人，却能幻作千万个影子。

我曾占有你，像一个美梦，在梦里称王，醒来只是一场空。

这无垠的宇宙对我都是虚幻；你才是，我的玫瑰，我全部财产。

小节标题：

1.我们为什么要听这本书？了解作为诗人的莎士比亚和他的内心世界
2.这本书讲了什么主要内容？莎士比亚的爱情观和人生观
3.我们能从这本书里学到什么？相信爱和诗具有永恒的生命和价值

《枕草子》

推广宣传语：一千多年前的微博和朋友圈
分类：诗与散文
国家：日本
作者：清少纳言

关于本书

“春天是破晓的时候最好。渐渐发白的山顶，有点亮了起来，紫色的云彩微细的横在那里，这是很有意思的。”这是日本平安时代的宫廷女官清少纳言在《枕草子》中的开篇名句，千百年来传颂不息。

《枕草子》是清少纳言将旧日在宫廷里的所见所闻整理而成的随笔集。作品由305个章节组成，大致分为类聚式章节、叙述感想式章节、日记回忆式章节。书中记录了平安时代奢华唯美的宫廷文化，也回忆了作者侍奉定子皇后的生活点滴，是研究日本传统文化和宫廷秘史的最佳读本之一。

关于作者

清少纳言（965—966）是日本平安时代著名的歌人、作家，中古三十六歌仙之一，与紫式部、和泉式部并称为平安时代的三大才女，曾任一条天皇皇后藤原定子身边的女官。

本书金句

  1. 春天是破晓的时候最好。渐渐发白的山顶，有点亮了起来，紫色的云彩微细的横在那里，这是很有意思的。

  2. 洗头，化妆，穿上浸满香气的衣裳，即使在没人看见的地方，心中也十分快活。

  3. 情人幽会，以夏季为宜。夜晚本来就短暂，不知不觉竟以至天明，所以往往不得就寝。到处敞开着，故亦得就便乘凉赏览庭景。心里仍有说不完的情话绵绵，彼此谈这谈那之间，鸟儿竟飞过眼前，又仿佛被人觑见秘密似的，颇饶情味。

  4. 嫉羡他人，抱怨己事，好蜚长流短，搬弄是非，哪怕是一丁点儿事情都要打破沙锅问到底，人家若是不愿相告，便口出怨言；又倘若知些许，便马上大大吹嘘，仿佛自己早晓得似的，讲个没完，这种人真是讨厌死了。

  5. 凡事物，不论草木鸟虫，且不管是辗转听闻，或偶有所感，皆不可漠不关心。

小节标题

1.我们为什么要读这本书？它是日本散文鼻祖，日本平安时代的文学双璧之一

2.这本书讲了什么主要内容？宫中七年的美好记忆，献给定子皇后的回忆录

3.我们能从这本书中学到什么？日本贵族女性敏感细腻的审美

《新月集·飞鸟集》

推广宣传语：每天读一句泰戈尔的诗，可以让我忘却世上一切苦痛
分类：诗与散文
国家：印度
作者：泰戈尔

关于本书

《新月集·飞鸟集》是泰戈尔的两部诗集合集。《新月集》于1913年出版，其中大部分内容以自己的童年经历和家庭生活为蓝本，描绘出一幅幅动人的美好图画；《飞鸟集》则更为精短，这部1916年完成的诗集共有325小节，每节短小精悍，智慧、沉思、哲学将是打开这部诗集的金钥匙。

关于作者

泰戈尔（1861—1941），印度家喻户晓的诗人、学者，著有《吉檀迦利》《新月集》《飞鸟集》《园丁集》《采果集》等诗集，同时他还写小说、散文和剧作，是一位文学创作的多面手。1913年，泰戈尔凭借《吉檀迦利》斩获诺贝尔文学奖。

本书金句

  1. 我独自在横跨过田地的路上走着，夕阳像一个守财奴似的，正藏起它的最后的金子。白昼更加深沉地投入黑夜之中。

  2. 我做云，你做月亮。我用两只手遮盖你，我们的屋顶就是青碧的天空。

  3. 不要因为你自己没有胃口而去责备你的食物。

  4. 水里的游鱼是沉默的，陆地上的兽类是喧闹的，空中的飞鸟是歌唱着的。但是人类却兼有了海里的沉默、地上的喧闹和空中的音乐。

  5. 你看不见你自己，你所看见的，只是你的影子。

小节标题

1.我们为什么要读这本书？诺贝尔文学奖第一次来到东方

2.《新月集》：一个活脱脱的儿童世界

3.《飞鸟集》：自然与情绪，智慧和爱

4.我们可以从这本书中获得什么？自由的诗学与美的召唤
